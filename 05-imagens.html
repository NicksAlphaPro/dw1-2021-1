Imagens

Tipos de imagens

1. Imagens vetoriais

Imagens vetoriaus são representadas internamente por equações matemáticas e formas geométricas em um sistema bidimensional de coordenadas.

Para exibir imagens veotorias, o computador precisa interpretar esseas equações e desnhar a imagem de acordo com as dimensões da imagem.

Vantegens

  • Podem ser redimensionais sem perda de qualidade.
  • São ideiais para representar ilustrações simples e desnhos técnicos, como mapas.

Desvantagens

  • Dependendo do nível de detalhe contido em uma imagem vetorial, o desenho pode demorar muito para carregar por conta do número de cálculos necessários para exíbi-la.

Formatos de Imagem vetorial

  • SVG (Scalable Vector Graphics, Figura de Vetores Escalávies). Esse formato é lido e gravado por uma série de programs, como Adoble Ilustrator e o Inskcape. Os navegadores de internet modernos são capazes de exibir SVG.
  • Oustro formatos de imagens vetorias, mas que não são suportados diretamente pelos navegadores de internet incluem: COR (Corel Draw!), AI Adoble Ilustrator)

2. Imagem de bitmap

São imagens que utilizam mapa de bits para represnetar a imagem. Uma imagem é interpretada como um mosaico de pontos (chamados pixles), e os formatos devem armazenar as informações de cor para cada um desses pontos.
